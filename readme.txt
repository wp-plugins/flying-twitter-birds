== Description ==

Twitter Bird flying animated effect by Javascript. When you'll scroll the page Up or Down, twitter bird will come in front of your page. Flying Twitter Bird found itself as the "follow me" link on the Twitter account. If you hover the bird, a "follow me" button and a "tweet this" button displayed.
